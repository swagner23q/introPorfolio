Why I Chose Programming


"Think of programming as a rubberband ball. Then you just keep adding to it more and more, and soon you have this massive rubberband ball of knowledge."

-Angelko Krstanovic, Software Engineer @Innovation Force

I chose programming because it has always been a passion of mine ever since I started to dive in through the years. I was able to open a source on a webpage and immediately find a specific code that I wanted to use for myself. Even when I was tinkering around with AOL Instant Messenger profiles, I created and customized for hours. I'd spend so much time playing with desktop UIs, such as the Enigma application. I've always felt like programming was something I could naturally do and figure out for hours. I'd be so determined to be 100% positive on a fix I would do to either a website or desktop interface. When I was younger, I had felt like if I pursued a career on a computer, I would be sitting down the rest of my life. Now, I see that no matter what pathway I try, it always comes back to programming.
